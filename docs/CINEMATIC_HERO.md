# Cinematic Hero Section Documentation

This documentation outlines the implementation of the "Velorah®" Cinematic Hero section, a high-end web experience built to showcase premium design and modern frontend technologies.

## 🚀 Technologies Used

- **Framework**: React 18
- **Build Tool**: Vite (Migrated from Create React App for performance)
- **Styling**: Tailwind CSS v3 (Utility-first styling)
- **Icons**: Lucide React
- **Animations**: CSS Keyframes + Tailwind Animation Utilities
- **Typography**: Google Fonts (Instrument Serif & Inter)

## ✨ Core Features

### 1. Fullscreen Video Background
A deep navy cinematic video loops in the background, providing visual depth without distracting from the content.
- **Source**: Cloudfront CDN
- **Properties**: Mutated, Looping, PlaysInline, Object-Cover

### 2. Liquid Glass Design
Custom glassmorphism effect implemented via the `.liquid-glass` CSS class.
- **Backdrop Blur**: 4px
- **Border Gradient**: A custom linear gradient mask that creates a "shimmering" edge effect.
- **Blend Mode**: Luminosity for a subtle integration with the video background.

### 3. Progressive Animations
Typography and buttons use a "Fade and Rise" animation sequence:
- **H1**: Immediate rise on load.
- **Subtext**: 200ms delay for a sequential feel.
- **CTA Button**: 400ms delay to guide the user's eye to the action.

## 🛠️ Maintenance & Customization

### Changing the Video
Update the `src` attribute in `src/components/Main.tsx`:
```tsx
<source src="YOUR_VIDEO_URL_HERE" type="video/mp4" />
```

### Adjusting Colors
Colors are defined in `src/index.css` using HSL variables. You can modify these to change the entire theme:
- `--background`: Deep navy blue (201 100% 13%)
- `--foreground`: Pure white (0 0% 100%)

### Updating Navigation
The navigation links are defined in `src/components/Main.tsx`. You can add or remove links in the `md:flex` container.
